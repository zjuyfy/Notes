CSS规则可以写在head里面style元素当中



CSS 规则由两个主要的部分构成：选择器{属性名:值},选择器通常是需要改变样式的 HTML 元素。

x1

选择器

CSS中id选择器以#开头，接id名（和链接一样）

CSS中class选择器以.开头，接class名

CSS中class选择器可以指定适用元素，写为元素名.类名

一个标签内多个class选择器可以用空格分开

外部样式表

写好CSS的外部文件后，在html中用link来引用样式：

<link rel="stylesheet" href="css文件(相对)位置">

优先级：内部style="" 高于 头部<style> 高于 外部样式文件
